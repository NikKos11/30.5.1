# 30.5.1
В директории /tests располагаются:

файл с тестами - 30_5_1_HW.py

файл settings.py - содержит информацию о валидном логинe и пароле. Дополнительно создан защищенный файл .env со значениями valid_login, valid_password (корректные данные логина и пароля пользоваталя для входа на сайт).

Если данный код выдал Вам ошибку "No module named ...", то необходимо дополнительно установить требуемые библиотеки. Для установки библиотек необходимо в Терминале Python прописать поочередно следующие строки и после нажать Enter:

   • pip install pytest (установит специальную библиотеку PyTest)
   • pip install pytest-selenium (установит специальную библиотеку pytest-selenium)
